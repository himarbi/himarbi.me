# React Portfolio - Ibrahim Abukar Ahmed

This repository contains my personal portfolio website, built with React, Vite, Chakra UI, and Framer Motion.

## Features
- **Responsive Design**: Designed to look great on desktop, tablet, and mobile.
- **Dark Mode**: Supports both light and dark themes using Chakra UI.
- **Interactive UI**: Smooth animations and transitions using Framer Motion.

## Setup
To run the project locally:
1. Clone the repository
2. Run \`npm install\`
3. Run \`npm run dev\`
